## File Specifications

### `src/lib/auth-client.ts`
```ts
import { createAuthClient } from 'better-auth/react';

export const authClient = createAuthClient({
  baseURL: process.env.NEXT_PUBLIC_API_URL,
});
```

### `src/lib/api.ts`
Axios instance:
- `baseURL`: `process.env.NEXT_PUBLIC_API_URL`
- `withCredentials: true` (for session cookie)

### `src/lib/query-client.ts`
TanStack Query client:
- `staleTime`: 60 seconds
- `retry`: 1

### `src/lib/utils.ts`
```ts
import { clsx } from 'clsx';
import { twMerge } from 'tailwind-merge';

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs));
}
```

### `src/providers/QueryProvider.tsx`
Wrap children in `<QueryClientProvider client={queryClient}>`.

### `src/providers/AuthProvider.tsx`
Expose session context from `authClient.useSession()` or similar better-auth hook. Make session available to child components.

### `src/middleware.ts`
- Protect `(customer)` routes: redirect unauthenticated → `/sign-in`
- Protect `(manager)` routes: redirect unauthenticated → `/sign-in`
- Redirect wrong role: MANAGER visiting customer routes → `/dashboard`; CUSTOMER visiting manager routes → `/explore`

### Root `src/app/layout.tsx`
- Font: Inter (via `next/font/google`)
- Wrap with `QueryProvider` and `AuthProvider`
- Include `<Toaster />` from sonner

### `src/app/globals.css`
Set CSS custom properties for shadcn theming and apply Inter font.

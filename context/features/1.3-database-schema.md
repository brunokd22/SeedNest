## Seed Script
**Location:** `apps/api/prisma/seed.ts`

Seed the following (use `bcryptjs` to hash passwords):
1. **Admin user** — `email: admin@seednest.com`, `role: ADMIN`
2. **Manager user** — any email, `role: MANAGER`
3. **Nursery** — linked to the manager user, with:
   - At least 2 sample `Category` records
   - At least 4 sample `Seedling` records spread across categories

Export a `main()` function and call it at the bottom of the file.

### `package.json` prisma config
```json
"prisma": {
  "seed": "tsx prisma/seed.ts"
}
```

## Commands to Run After Implementation
```bash
npx prisma generate
npx prisma migrate dev --name init
npx prisma db seed
```

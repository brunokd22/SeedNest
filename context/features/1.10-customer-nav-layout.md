Build the Customer layout with sticky top navigation, cart store, and role guard.

---

## `cart-store.ts`

### Cart Item Shape
```ts
type CartItem = {
  seedlingId:  string;
  name:        string;
  price:       number;
  quantity:    number;
  nurseryId:   string;
  photo:       string;
};
```

### Store Interface
```ts
type CartStore = {
  items: CartItem[];

  // Actions
  addItem:        (item: CartItem) => void;
  removeItem:     (seedlingId: string) => void;
  updateQuantity: (seedlingId: string, quantity: number) => void;
  clearCart:      () => void;

  // Computed (derived via selectors or getters)
  totalItems:  () => number;    // sum of all item quantities
  totalAmount: () => number;    // sum of price × quantity for all items
};
```

### Persistence
Use `zustand/middleware` `persist` middleware, storing to `localStorage` under the key `'seednest-cart'`.

## `CustomerNav.tsx`

### Layout
Sticky top bar: `position: sticky`, `top: 0`, `z-50`, white background, `shadow-sm`.

### Desktop Layout
```
[ SeedNest logo ]    [ Explore | My Orders | My Issues ]    [ CartBadge | UserMenu ]
```

### Mobile Layout
```
[ SeedNest logo ]                                           [ CartBadge | Hamburger ]
```
Hamburger opens a shadcn **Sheet** containing all nav links stacked vertically.

### Navigation Links
| Label | Route |
|---|---|
| Explore | `/explore` |
| My Orders | `/my-orders` |
| My Issues | `/my-issues` |

Active link styling: `text-primary font-medium`

### `CartBadge` (inline component or sub-component)
- Icon: `ShoppingCart` (lucide-react)
- Reads `totalItems()` from `cart-store`
- When `totalItems > 0`: show a red circle badge (absolute positioned, top-right of icon) with the count
- Links to `/cart`

### `UserMenu`
shadcn `DropdownMenu` with `Avatar` trigger (initials fallback):
- My Orders → `/my-orders`
- My Issues → `/my-issues`
- Sign Out → `authClient.signOut()` then `router.push('/sign-in')`

## `(customer)/layout.tsx`


### Role Guard
```ts
const session = await auth.getSession(); // server-side
if (session?.user.role !== 'CUSTOMER') redirect('/dashboard');
```

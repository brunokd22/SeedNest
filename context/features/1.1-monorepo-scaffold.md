# Spec 1.1 — Monorepo Scaffold

## Goal
Create the Turborepo monorepo root and shared packages. Do **not** scaffold any app yet — only the root config and `packages/`.

---

## Workspace Structure to Create

```
seednest/
  apps/                        ← empty for now (dirs only)
    api/
    web/
    mobile/
  packages/
    shared/
      src/
        index.ts               ← export {}
      package.json
    tsconfig/
      base.json
      nextjs.json
      react-native.json
  turbo.json
  package.json                 ← root
  .gitignore
  .prettierrc
  tsconfig.json                ← root (extends packages/tsconfig/base.json)
```

---

## File Specifications

### `turbo.json`
Define pipelines for: `build`, `dev`, `lint`, `type-check`.

### Root `package.json`
- Workspaces: `apps/*`, `packages/*`
- DevDependencies: `turbo`, `typescript`, `prettier`, `eslint`

### `.gitignore`
Cover: `node_modules`, `.turbo`, `.next`, `dist`, `.expo`, `*.env`

### `.prettierrc`
```json
{
  "singleQuote": true,
  "trailingComma": "all",
  "semi": true,
  "printWidth": 100
}
```

### `packages/tsconfig/base.json`
- Strict mode
- Target: `ES2022`
- `moduleResolution`: `bundler`
- `noEmit`: true

### `packages/tsconfig/nextjs.json`
- Extends `base.json`
- Adds `"jsx": "preserve"`

### `packages/tsconfig/react-native.json`
- Extends `base.json`
- Adds `"jsx": "react-native"`

### `packages/shared/src/index.ts`
```ts
export {};
```

### `packages/shared/package.json`
- `name`: `"@seednest/shared"`
- Main export points to `src/index.ts`

### Root `tsconfig.json`
Extends `packages/tsconfig/base.json`.

---

## Completion Criteria
- [ ] `pnpm install` succeeds at root
- [ ] No TypeScript errors in `packages/`
- [ ] `packages/tsconfig/` exports three config files
- [ ] `packages/shared` builds without errors

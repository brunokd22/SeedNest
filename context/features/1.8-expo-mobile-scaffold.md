## File Specifications

### `lib/auth.ts`
Token storage using `expo-secure-store`:
```ts
export async function getToken(): Promise<string | null>
export async function setToken(token: string): Promise<void>
export async function clearToken(): Promise<void>
```

### `lib/api.ts`
Axios instance:
- `baseURL`: from env / constants (e.g. `http://localhost:4000`)
- Request interceptor: attach `Authorization: Bearer <token>` from SecureStore on every request

### `lib/query-client.ts`
TanStack Query client with `staleTime: 60_000`, `retry: 1`.

### `store/auth-store.ts`
Zustand store:
```ts
type AuthStore = {
  user: AuthUser | null;
  token: string | null;
  setAuth: (user: AuthUser, token: string) => void;
  clearAuth: () => void;
};
```

### `providers/AuthProvider.tsx`
On mount:
1. Read token from SecureStore
2. If token exists, call `GET /api/auth/session` to validate
3. On success: populate `auth-store` with user + token
4. On failure: clear token and redirect to sign-in

### `providers/QueryProvider.tsx`
Wrap children in `<QueryClientProvider client={queryClient}>`.

## Auth Screens

### `(auth)/sign-up.tsx`
- `KeyboardAvoidingView` + `ScrollView`
- Fields: Full Name, Email, Password (show/hide toggle)
- Role selector: two tappable cards — "I'm a Manager" / "I'm a Customer"
  - Selected card: border `#2D6A4F`, filled background
- On submit: `POST /api/auth/sign-up/email`
- On success: navigate to `/(auth)/verify-email` passing `email` as param

### `(auth)/sign-in.tsx`
- Fields: Email, Password
- On submit: `POST /api/auth/sign-in/email`
- On success: store token via `setToken()` + `setAuth()` in zustand → navigate to `/(tabs)/explore`
- Link to `/(auth)/forgot-password`

### `(auth)/forgot-password.tsx`
Mirror web app logic using React Native components (no browser-specific APIs).

### `(auth)/verify-email.tsx`
Mirror web app OTP logic:
- 6 individual `TextInput` boxes, `maxLength={1}`
- Auto-focus next on input, auto-focus previous on backspace
- On submit: `POST /api/auth/verify-email`
- Resend button with 60-second cooldown
